# EduLink Network — Captive Portal

A WiFi monetization portal built for university students in MAU Yola, Adamawa Sate, Nigeria.
Powered by Starlink satellite internet.

## Features
- Hostel-based session management
- Multiple internet plans (Day, Night, Weekly)
- Paystack payment integration
- Voucher code validation
- Real-time transaction logging
- Admin dashboard with per-hostel analytics
- Session recovery system

## Tech Stack
- HTML, CSS, JavaScript
- Supabase (database)
- Paystack (payments)
- MikroTik (hotspot management)
